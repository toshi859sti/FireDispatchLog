# 消防団記録

消防団の団員管理と出動記録を管理するAndroidアプリケーション

## 機能

### 団員名簿管理
- 最大20名の団員情報（氏名・電話番号）を管理
- 五十音順での自動ソート
- 役職割り当て機能

### 役職管理
- 15種類の役職（分団長、副分団長、消防部長、警護部長、給水班長、機械班長、火先班長、救護班長、警交班長、給水団員、機械団員、火先団員、救護団員、警交団員、補助団員）
- 一般団員数設定（合計11名）
- 自動バリデーション

### 出動表
- グリッド形式での出動記録管理
- 出動回数・手当指数の自動集計
- 今年度フィルタリング機能
- PDF出力（A4横向き、複数ページ対応）

### データ管理
- JSONファイルでのバックアップ・復元
- 会計年度設定（4月〜翌3月）
- 分団名設定
- データ初期化

## 技術スタック

- **言語**: Kotlin
- **UI**: Jetpack Compose
- **デザイン**: Material Design 3
- **データベース**: Room
- **ナビゲーション**: Navigation Compose
- **アーキテクチャ**: MVVM

## 要件

- Android 8.0 (API 26) 以上
- 開発環境: Android Studio Ladybug (2024.2.1) 以降

## ビルド方法

1. Android Studioでプロジェクトを開く
2. Gradle Syncを実行
3. エミュレータまたは実機で実行

## 使い方

### 初期設定
1. メニュー画面から「団員名簿」を開く
2. 「名簿編集」で団員情報を入力
3. 「役職割り当て」で一般団員数を設定（合計11名）
4. 各団員に役職を割り当て

### 出動記録
1. メニュー画面から「出動表」を開く
2. 「追加」で行事を登録
3. 日付・行事名・手当指数・出席者を入力
4. 集計は自動で計算されます

### PDF出力
1. 出動表画面で「PDF出力」をタップ
2. 保存先を選択
3. PDFファイルが生成されます

### バックアップ
1. メニュー画面から「設定」を開く
2. バックアップセクションで「エクスポート」をタップ
3. 保存先を選択してJSONファイルを保存

## 注意事項

- 団員は最大20名まで登録可能
- 一般団員の合計は必ず11名に設定する必要があります
- 補助団員は出動表には表示されません
- PDF出力時は分団名を設定することを推奨します
